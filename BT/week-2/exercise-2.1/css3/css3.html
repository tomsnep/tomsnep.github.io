Some text

The fallback for text-decoration styling is quite simple. By making a <div> container around the text element and adding a empty <div> under the <p> you can simulate the dotted green line with css

Html:
<div class="text-decoration-fallback-container">
<p>Some text</p>
<div></div>
</div>

Css:
.text-decoration-fallback-container {
width: max-content;
width: -moz-max-content;
}
.text-decoration-fallback-container > p {
margin-bottom: 0;
}
.text-decoration-fallback-container > div {
width: 100%;
border-bottom: 1px dotted green;
}
